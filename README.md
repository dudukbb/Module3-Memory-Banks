# Module 03 — Context Engineering: Memory Banks

This repository contains the deliverables for the EPAM AI Tech Bootcamp Module 03 lab focused on AI context engineering and memory bank design.

The project demonstrates how structured memory banks improve AI-generated outputs by providing persistent architectural, domain, workflow, and coding-standard context.

---

## Project Overview

**Project Name:** Personal Task Board (mrello)

mrello is a lightweight real-time task management platform designed for small engineering teams.  
The memory banks created in this lab provide reusable AI context for:
- architecture decisions
- coding conventions
- domain terminology
- workflows
- security and testing expectations

The repository focuses on specification-driven development and AI-assisted engineering workflows rather than full application implementation.

## AI Context Engineering Goals

This lab evaluates how memory banks improve AI-assisted development workflows by:
- reducing generic AI output
- improving architectural consistency
- enforcing project conventions
- improving security alignment
- reducing correction and refactoring time

---

## Memory Bank Impact Analysis

The repository includes a before/after comparison showing the difference between:
- AI generation without memory banks
- AI generation with structured project context

Measured improvements include:
- better architecture alignment
- improved validation and security handling
- standardized API responses
- stronger testing scaffolding
- reduced correction time
